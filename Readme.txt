Follow this steps to run the web application:

1. open a terminal and execute these commands:
- npm i
- npm i @remotion/google-fonts
- cd server
- npm start

2. open another terminal and execute these commands:
- cd frontend
- npm run dev
if terminal says 'vite'is not recognized 
- npm i -g vite or npm i vite,then run again
- Ctrl + click the link to open the web application
